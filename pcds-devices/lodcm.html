LODCM¶

class pcdsdevices.epics.LODCM(prefix, *, ioc='', read_attrs=None, name=None, **kwargs)¶
destination(line=None)¶

Return where the light is going for a given line at the current LODCM state.

Parameters:line (str or int, optional) – The starting line for the beam. If this is 1 or “MAIN”, light is coming in on H1N from the main line. If this is 2 or “MONO”, light is coming in on H2N to the mono line (e.g. XCS periscope)
Returns:destination – “BLOCKED” if the light (probably) does not go through. “MAIN” if the light continues on the main line. “MONO” if the light continues on the mono line. “BOTH” if the light continues on both lines. “UNKNOWN” if the state is strange.
Return type:str
